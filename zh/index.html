在网站中使用颠倒文字

您可以使用CSS在自己的网站中创建颠倒文字效果：

上下颠倒效果

.upside-down {
  display: inline-block;
  transform: rotate(180deg);
  white-space: nowrap;
}

水平镜像效果

.horizontal-mirror {
  display: inline-block;
  transform: scaleX(-1);
}

垂直镜像效果

.vertical-mirror {
  display: inline-block;
  transform: scaleY(-1);
  white-space: nowrap;
}

HTML 示例

<span class="upside-down">颠倒的文字</span>
